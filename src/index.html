PacBomb

KAROUIA ALAEDINE & GARNIER CORENTIN

CONCEPT DU JEU: Vous etes un pacman et des fantomes vous poursuivent, vous avez comme seul moyen de défense des bombes et des bonus, votre but est de survivre le plus longtemps possible sans vous faire toucher par les fantomes ni par vos bombes, vous n'avez qu'une seule chance.

EVENEMENTS: Les fantomes apparaitront a un certain rythme aleatoire dans le labyrinthe.
Le labyrinthe est composé de 3 differents elements.

  • il y a le chemin de base ou le joueur se deplace
  • les arbres qui sont destructibles
  • les blocs de pierres qui sont indestructibles
Des Bonus apparaissent dans le labyrinthe de maniere aleatoirement. Ils en existent 3 avec leur effet associé:
  • rouge -> augmentation du nombre de bombe posable 3 à 5;
  • noir -> invincibilité durant une courte periode;
  • bleu -> les fantomes sont gelés

Les CONTROLES: Le joueur se déplace à l'aide des flèches directionnelles. Les bombes se posent avec la touche espace du clavier.
